# Grading Rubric

| Category | Points | Evidence |
|---|---:|---|
| Circuit breaker and fallback | 25 | state transitions, route reasons, no retry storm |
| Cache and cost | 20 | TTL/threshold, hit rate, cost saved, false-hit examples |
| Observability and metrics | 20 | metrics JSON/CSV, P50/P95/P99, counters/gauges |
| Chaos/load testing | 20 | 3+ scenarios, recovery evidence, concurrent load |
| Report and code quality | 15 | README, configs, tests, type hints, report clarity |
